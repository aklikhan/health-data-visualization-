\# Health Data Visualization



Interactive scatterplot visualization of health survey data using D3.js.



\## Features

\- Interactive scatterplot with dynamic axis selection

\- Hover tooltips for detailed information

\- Point selection and highlighting

\- Animated transitions

\- Responsive design



\## Live Demo

\[View the live visualization](https://YOUR\_USERNAME.github.io/health-data-visualization/)



\## Local Setup

1\. Clone this repository

2\. Open `index.html` in a web browser

3\. No additional dependencies required



\## Technologies

\- HTML5, CSS3, JavaScript

\- D3.js v7

\- GitHub Pages for deployment



\## Dataset

Health survey data containing demographic and health-related variables.
